Тема 3. Коды представления чисел

В вычислительной технике для работы с положительными и отрицательными числами используются специальные коды:

  • прямой;
  • обратный;
  • дополнительный;
  • модифицированный обратный;
  • модифицированный дополнительный;
  • код со смещением.

Прямой код (на примере 2-ичной СС)

Образуется из записи модуля числа в двоичной системе путем добавления старшего разряда, который отводится для кодирования знака. Для двоичной системы, если в знаке: 0 – число положительное; 1 – отрицательное. (В других П-Т СС максимальная цифра в знаковом разряде является признаком отрицательного числа)

На бумаге знаковый разряд отделяется точкой.

1110 = 10112=0.10112 пр.к.

-1110 = 10112=1.10112 пр.к.

  • Вычисления в прямом коде требуют отдельной аппаратуры для сложения и вычитания.
  • Вычисления производятся над модулями чисел по правилам двоичной системы.
  • Знаковые разряды операндов обрабатываются отдельно, чтобы определить знак результата. При вычитании меньшее значение модуля вычитается из большего, а знак разности определяется по знаку большего по модулю числа.
  • Необходимо следить за переполнением разрядной сетки, которое может возникнуть, если слагаемые и результат должны иметь одинаковую разрядность.

Затраты аппаратуры, необходимые для выполнения сложения и вычитания в прямом коде на примере четырехразрядных двоичных чисел:

Вычитатель более чем на 60% сложнее сумматора!

Использование обратных и дополнительных кодов позволяет свести операцию вычитания к операции сложения и отказаться от использования вычитателя. За счет этого достигается заметная экономия аппаратных средств.

Обратный код. Общая идея.

Исторический предшественник дополнительного кода. Использовался в арифмометрах и первых ЭВМ: CDC 6600, LINC, PDP-1 и UNIVAC 1107. Является промежуточным этапом при переводе в дополнительный код.

  • Положительные числа в обратном коде выглядят также как и в прямом.
  • Отрицательные числа представляются следующим образом: в знаковый разряд ставится 1, остальные разряды инвертируются.
  • Обратный код N-разрядного отрицательного числа является дополнением модуля этого числа до максимального числа, которое можно записать при помощи N разрядов.

При этом знак отрицательного числа в обратном коде кодируется максимальной цифрой используемой системы счисления и располагается в старшем разряде.

-28=7.58 обр. код

1110=10112=0.10112 пр.код=0.10112 обр. код

-1110=-10112=1.10112 пр.код=1.01002 обр. код

Правила сложения

  • Перевести слагаемые в обратный код.
  • Выровнять разрядности слагаемых, добавив необходимое количество незначащих разрядов к более короткому слагаемому (можно делать еще на этапе прямого кода).
  • Произвести сложение по правилам используемой системы счисления. Знаковые разряды участвуют в сложении.
  • Единицу переноса из знакового разряда прибавить к младшему разряду суммы.
  • Проверить наличие переполнения.

Признаком переполнения (в двоичной системе) является получение отрицательного результата при сложении положительных слагаемых и наоборот. При обнаружении переполнения для решения задачи необходимо (если это возможно) увеличить разрядную сетку хотя бы на один разряд и повторить вычисления.

Дополнительный код. Общая идея.

Определение

  • Положительные числа выглядят так же, как и в прямом коде
  • В знаковый разряд отрицательных чисел ставится единица, далее берется число в обратном коде, и к младшему разряду арифметическим образом прибавляется единица.

-1410=1.00102 доп. -810=1.10002 доп.

1110=10112=0.10112 пр. код=0.10112 доп. код

Для отрицательных чисел разряды двоичной записи модуля числа инвертируются и к младшему разряду прибавляется 1. В знаковом разряде записывается 1.

-1110=-10112=1.10112 пр. код=1.0100+1=1.01012 доп. код

Альтернативный способ перевода. Нужно переписать все биты модуля исходного двоичного числа справа налево до первой единицы, включая ее. Остальные биты инвертировать. Поставить знаковый разряд.

Действия по переводу отрицательного числа из дополнительного кода в прямой точно такие же, как и при переводе из прямого кода в дополнительный.

1.10102 доп. код=>1.0101+1=>1.01102 пр. код = -610

  • Есть только +0
  • Дополнительный код N-разрядного отрицательного числа есть дополнение модуля этого числа до 0.
  • Есть невостребованная комбинация (-8 требует уже 5 бит).

Правила сложения

  • Перевести слагаемые в дополнительный код
  • Выровнять разрядности слагаемых, добавив необходимое количество незнаяащих разрядов к более короткому слагаемому
  • Произвести сложение по правилам пользуемой системы счисления. Знаковые разряды участвуют в сложении
  • Единицу переноса из знакового разряда отбросить
  • Проверить наличие переполнения

Модифицированные коды

Для придания однозначности записи числа могут использоваться модифицированные обратные и дополнительные коды

В модифицированных кодах используются два знаковых разряда

  • 00 - положительное число;
  • 11 - отрицательное число;
  • 01 - положительное число с переполнением;
  • 10 - отрицательное число с переполнением

Правила вычислений в модифцированных кодах такие же

Код со смещением (несимметричный)

Позволяет сдвинуть числовую шкалу, содержащую как отрицательные, так и положительные числа, полностью в область положительных чисел. Представление положительных и отрицательных чисел несимметрично относительно нуля.

Код со смещением (симметричный)

Можно сделать симметричное относительно нуля кодирование, если одну комбинацию не использовать для записи чисел или использовать ее для специальных нужд. (Например, в стандарте IEEE754 для обозначения бесконечности)

Если n – доступное количество разрядов, то:

  • 2n-1-1 - максимальное число, которое можно записать в смещенном коде как 1..10;
  • -2n-1+1 - минимальное число, которое можно записать в смещенном коде как 0…0;
  • 2n-1-1 - величина смещения

При выполнении арифметических операций необходимо учитывать смещение
